## 项目概述

StockHub 是一个**多券商智能集中管理站**的产品设计提案与交互 Demo。核心理念：用户无需在平台开户，身份证验证后手动绑定已有券商账户（支持21家券商），平台充当中央管理驾驶舱，实现跨券商持仓聚合、一键切换交易。

**四大差异化能力：**
- 🏦 **多券商集中管理** — 身份证验证 + 手动绑定，不要求开户
- 🤖 **AI全功能对话助手** — 10类意图：行情/持仓/交易/资讯/推荐/账户/市场/看守/交易分析/经验学习
- 🔔 **AI看门狗** — 自然语言创建条件任务，后台自动监控触发提醒/下单
- 🧠 **交易行为学习** — 6维度个人洞见 + 散户常见错误 + 机构手法对比

## 五因子分析模型

每只股票从5个维度综合评分（满分100），每个因子拆3个子项：

| 因子 | 权重 | 子项拆解 | 说明 |
|------|:----:|----------|------|
| 趋势动量 | 30 | 短期趋势 · 中期趋势 · 长期趋势 | 多周期均线排列 |
| 估值合理 | 20 | PE历史分位 · PB历史分位 · 股息率 | 相对估值位置 |
| 波动风控 | 15 | 年化波动率 · 最大回撤 · Beta系数 | 风险度量 |
| 量能资金 | 20 | 量比 · 资金流向 · 换手率 | 量价配合 |
| 技术形态 | 15 | MACD · KDJ · 布林带 | 经典技术指标 |
