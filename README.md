# StockHub · 多券商智能集中管理站

> 身份证验证 → 21家券商绑定 → 集中管理
> AI全功能对话助手：行情查询 · 持仓分析 · 交易复盘 · 经验学习
> AI看门狗自动交易 · 策略回测 · 多券商一键切换

---

## 产品定位

**不要求用户开户**，身份证验证后手动绑定已有券商账户。

| 特性 | 说明 |
|------|------|
| 🆔 **身份证验证** | SHA256哈希存储，仅做实名认证 |
| 🏦 **21家券商支持** | 东北/中信/华泰/国泰/招商/广发/海通… |
| 📋 **手动绑定** | 录入已有账号，无需在本平台开户 |
| 📊 **集中管理** | 按券商分组展示，一键选择执行交易 |

> 核心理念：用户无需新开证券账户 → 只需身份证验证 → 手动绑定已有券商账号 → 平台充当中央管理驾驶舱 → 一键在多券商间切换交易

## 使用场景全路径

### ☀️ 早盘场景 (09:30)
1. AI助手问「大盘怎么样」→ 市场概况
2. 看行情Tab涨幅榜 → 发现金山办公+3.2%
3. AI助手问「金山办公股价」→ 看AI评分✦92 → 设置跌破360元提醒我

### 📊 盘中交易场景 (10:00-15:00)
1. 交易Tab → 选择券商(东北证券) → 金山办公
2. 买入200股 → 确认 → 成交
3. 看门狗触发：「宁德时代跌破190元」→ 自动买入200股 → 弹窗通知

### 🌙 收市复盘场景 (15:30+)
1. AI助手问「分析我的交易习惯」→ 盈亏仪表盘+行为洞见
2. AI助手问「散户常见错误」→ 对比学习·发现自己持仓过于集中
3. AI助手问「机构如何操作」→ 学习北向资金趋势跟随手法
4. 持仓Tab → 看4维KPI + AI选股推荐

### ⚙️ 配置管理场景
1. 「我的」Tab → 绑定新券商(华泰证券) → 手动录入已有持仓
2. AI助手 → 创建多条看守任务 → 设置自动交易规则
3. 查看已触发任务历史 → 统计胜率 → 优化策略
4. AI助手问「我的交易习惯」→ 获取阶段性总结与改进建议
